COSY PROGRESS CLUB — VERSION 1

WHAT IS INCLUDED
- Today page with daily goals, difficulty points, day type, agenda, cycle status, and gentle substitutions
- Sunday-start weekly planner with copying and moving flexible goals
- Gentle Zone 2 cardio logging and progression
- Academic projects plus optional focused-minute logging
- No-spend, transit-only, and spending-day tracking
- Multiple savings jars
- Vanessa coin rewards and shop
- Summary statistics
- Local saving in your browser
- Export/import backup

HOW TO USE ON IPHONE
1. Unzip the folder.
2. Put the folder on a web host or open index.html from a Files-compatible web server.
3. Open the site in Safari.
4. Tap Share.
5. Tap “Add to Home Screen.”
6. Keep the same browser/site storage so progress remains available.

IMPORTANT
This version saves progress locally in the browser. Use “Export progress backup” regularly.
Google Calendar and Apple Health are not connected in Version 1.
